# Solutie:
```
    - am cautat prima zi in care s-a modificat bugetul
    - daca bugetul e > 0 generez cost in ziua respectiva
        in intervalul de timp in care s-a modificat bugetul
    - la fiecare pas verific costul generat in fiecare zi nu 
        depaseste bugetul zilnic sau lunar
```

# Instalare:
```
    1. git clone https://github.com/iulianbeleiu/adwords
    2. Install docker and docker-compose
    3. cd adwords/docker
    4. docker-compose up
    It will bring up MariaDB + Php-fpm + Nginx
    The website should be running at http://localhost
```
